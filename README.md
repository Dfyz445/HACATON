### Реализованная функциональность:
- **Форма сбора заявок:** Интерактивный смарт-квиз для сбора контактных данных, предпочтений по стилю, бюджету, типу помещения и зонам.
- **Обработка данных:** Валидация введённых пользователем данных (например, проверка имени на наличие цифр).
- **Сохранение в БД:** Отправка анкеты в базу данных с привязкой к зонам, стилю, бюджету и типу помещения.

### Особенность проекта в следующем:

- **Унифицированная структура данных:** Все модули используют согласованный формат полей (например, name, phone, email), что обеспечивает стабильную работу бэкенда.
- **Поддержка связей many-to-many:** Реализована привязка выбранных пользователем зон к заявке через связующую таблицу leads_has_zones.
- **Надёжность и безопасность:** Используется транзакционная модель работы с БД, автоматический rollback при ошибках и защита от SQL-инъекций через параметризованные запросы.

# Установка 
## База данных

1. Вход в консоль MySQL
```
mysql -u root -p
```
2. Создание базы с правильной кодировкой
```
CREATE DATABASE mydb CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
```
3. Импорт структуры и данных файла
```
USE mydb;
SOURCE /путь/к/файлу/database.sql;
```
4. Подключится по следующим параметрам:
Host: 127.0.0.1
